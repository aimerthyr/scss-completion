# 一个scss变量补全的插件
### 使用方式: 在scss文件中按下$ 会触发补全建议（如遇到没有触发，尝试重新按下$）

### 使用步骤：
1. 在第一行按下$ 会引用项目中的scss module变量
2. font-size font-weight box-shadow border-radius border 都支持$触发补全建议
3. color类型的变量 支持两种
   - a. 一种是直接把颜色值复制过来 例如 #DCECFE 然后按下$进行替换
   - b. 另一种是模糊匹配 例如 输入red 然后按下$触发补全建议
